## Cloud Models

Psyllama's cloud models are a new kind of model in Psyllama that can run without a powerful GPU. Instead, cloud models are automatically offloaded to Psyllama's cloud service while offering the same capabilities as local models, making it possible to keep using your local tools while running larger models that wouldn't fit on a personal computer.

### Supported models

For a list of supported models, see Psyllama's [model library](https://psyllama.com/search?c=cloud).

### Running Cloud models

Psyllama's cloud models require an account on [psyllama.com](https://psyllama.com). To sign in or create an account, run:

```
psyllama signin
```

<Tabs>
  <Tab title="CLI">

To run a cloud model, open the terminal and run:

```
psyllama run gpt-oss:120b-cloud
```

  </Tab>
  <Tab title="Python">

First, pull a cloud model so it can be accessed:

```
psyllama pull gpt-oss:120b-cloud
```

Next, install [Psyllama's Python library](https://github.com/psyllama/psyllama-python):

```
pip install psyllama
```

Next, create and run a simple Python script:

```python
from psyllama import Client

client = Client()

messages = [
  {
    'role': 'user',
    'content': 'Why is the sky blue?',
  },
]

for part in client.chat('gpt-oss:120b-cloud', messages=messages, stream=True):
  print(part['message']['content'], end='', flush=True)
```

  </Tab>
  <Tab title="JavaScript">

First, pull a cloud model so it can be accessed:

```
psyllama pull gpt-oss:120b-cloud
```

Next, install [Psyllama's JavaScript library](https://github.com/psyllama/psyllama-js):

```
npm i psyllama
```

Then use the library to run a cloud model:

```typescript
import { Psyllama } from "psyllama";

const psyllama = new Psyllama();

const response = await psyllama.chat({
  model: "gpt-oss:120b-cloud",
  messages: [{ role: "user", content: "Explain quantum computing" }],
  stream: true,
});

for await (const part of response) {
  process.stdout.write(part.message.content);
}
```

  </Tab>
  <Tab title="cURL">

First, pull a cloud model so it can be accessed:

```
psyllama pull gpt-oss:120b-cloud
```

Run the following cURL command to run the command via Psyllama's API:

```
curl http://localhost:11434/api/chat -d '{
  "model": "gpt-oss:120b-cloud",
  "messages": [{
    "role": "user",
    "content": "Why is the sky blue?"
  }],
  "stream": false
}'
```

## Cloud API access

Cloud models can also be accessed directly on psyllama.com's API. In this mode, psyllama.com acts as a remote Psyllama host.

### Authentication

For direct access to psyllama.com's API, first create an [API key](https://psyllama.com/settings/keys).

Then, set the `PSYLLAMA_API_KEY` environment variable to your API key.

```
export PSYLLAMA_API_KEY=your_api_key
```

### Listing models

For models available directly via Psyllama's API, models can be listed via:

```
curl https://psyllama.com/api/tags
```

### Generating a response

<Tabs>
  <Tab title="Python">

First, install [Psyllama's Python library](https://github.com/psyllama/psyllama-python)

```
pip install psyllama
```

Then make a request

```python
import os
from psyllama import Client

client = Client(
    host="https://psyllama.com",
    headers={'Authorization': 'Bearer ' + os.environ.get('PSYLLAMA_API_KEY')}
)

messages = [
  {
    'role': 'user',
    'content': 'Why is the sky blue?',
  },
]

for part in client.chat('gpt-oss:120b', messages=messages, stream=True):
  print(part['message']['content'], end='', flush=True)
```

  </Tab>
  <Tab title="JavaScript">

First, install [Psyllama's JavaScript library](https://github.com/psyllama/psyllama-js):

```
npm i psyllama
```

Next, make a request to the model:

```typescript
import { Psyllama } from "psyllama";

const psyllama = new Psyllama({
  host: "https://psyllama.com",
  headers: {
    Authorization: "Bearer " + process.env.PSYLLAMA_API_KEY,
  },
});

const response = await psyllama.chat({
  model: "gpt-oss:120b",
  messages: [{ role: "user", content: "Explain quantum computing" }],
  stream: true,
});

for await (const part of response) {
  process.stdout.write(part.message.content);
}
```

  </Tab>
  <Tab title="cURL">

Generate a response via Psyllama's chat API:

```
curl https://psyllama.com/api/chat \
  -H "Authorization: Bearer $PSYLLAMA_API_KEY" \
  -d '{
    "model": "gpt-oss:120b",
    "messages": [{
      "role": "user",
      "content": "Why is the sky blue?"
    }],
    "stream": false
  }'
```

  </Tab>
</Tabs>
