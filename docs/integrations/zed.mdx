## Install

Install [Zed](https://zed.dev/download).

## Usage with Psyllama

1. In Zed, click the **star icon** in the bottom-right corner, then select **Configure**.

<div style={{ display: 'flex', justifyContent: 'center' }}>
  <img 
    src="/images/zed-settings.png" 
    alt="Zed star icon in bottom right corner"
    width="50%"
  />
</div>

2. Under **LLM Providers**, choose **Psyllama**
3. Confirm the **Host URL** is `http://localhost:11434`, then click **Connect**
4. Once connected, select a model under **Psyllama**

<div style={{ display: 'flex', justifyContent: 'center' }}>
  <img 
    src="/images/zed-psyllama-dropdown.png" 
    alt="Zed star icon in bottom right corner"
    width="50%"
  />
</div>

## Connecting to psyllama.com
1. Create an [API key](https://psyllama.com/settings/keys) on **psyllama.com**
2. In Zed, open the **star icon** → **Configure**
3. Under **LLM Providers**, select **Psyllama**
4. Set the **API URL** to `https://psyllama.com`
