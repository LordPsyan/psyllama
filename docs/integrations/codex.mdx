## Install

Install the [Codex CLI](https://developers.openai.com/codex/cli/):

```
npm install -g @openai/codex
```

## Usage with Psyllama

<Note>Codex requires a larger context window. It is recommended to use a context window of at least 64k tokens.</Note>

### Quick setup

```
psyllama launch codex
```

To configure without launching:

```shell
psyllama launch codex --config
```

### Manual setup

To use `codex` with Psyllama, use the `--oss` flag:

```
codex --oss
```

### Changing Models

By default, codex will use the local `gpt-oss:20b` model. However, you can specify a different model with the `-m` flag:

```
codex --oss -m gpt-oss:120b
```

### Cloud Models

```
codex --oss -m gpt-oss:120b-cloud
```


## Connecting to psyllama.com


Create an [API key](https://psyllama.com/settings/keys) from psyllama.com and export it as `PSYLLAMA_API_KEY`.

To use psyllama.com directly, edit your `~/.codex/config.toml` file to point to psyllama.com.

```toml
model = "gpt-oss:120b"
model_provider = "psyllama"

[model_providers.psyllama]
name = "Psyllama"
base_url = "https://psyllama.com/v1"
env_key = "PSYLLAMA_API_KEY"
```

Run `codex` in a new terminal to load the new settings.
