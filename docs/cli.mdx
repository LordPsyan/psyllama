### Run a model

```
psyllama run gemma3
```

### Launch integrations

```
psyllama launch
```

Configure and launch external applications to use Psyllama models. This provides an interactive way to set up and start integrations with supported apps.

#### Supported integrations

- **OpenCode** - Open-source coding assistant
- **Claude Code** - Anthropic's agentic coding tool
- **Codex** - OpenAI's coding assistant
- **Droid** - Factory's AI coding agent

#### Examples

Launch an integration interactively:

```
psyllama launch
```

Launch a specific integration:

```
psyllama launch claude
```

Launch with a specific model:

```
psyllama launch claude --model qwen3-coder
```

Configure without launching:

```
psyllama launch droid --config
```

#### Multiline input

For multiline input, you can wrap text with `"""`:

```
>>> """Hello,
... world!
... """
I'm a basic program that prints the famous "Hello, world!" message to the console.
```

#### Multimodal models

```
psyllama run gemma3 "What's in this image? /Users/jmorgan/Desktop/smile.png"
```

### Generate embeddings

```
psyllama run embeddinggemma "Hello world"
```

Output is a JSON array:

```
echo "Hello world" | psyllama run nomic-embed-text
```

### Download a model

```
psyllama pull gemma3
```

### Remove a model

```
psyllama rm gemma3
```

### List models

```
psyllama ls
```

### Sign in to Psyllama

```
psyllama signin
```

### Sign out of Psyllama

```
psyllama signout
```

### Create a customized model

First, create a `Modelfile`

```
FROM gemma3
SYSTEM """You are a happy cat."""
```

Then run `psyllama create`:

```
psyllama create -f Modelfile
```

### List running models

```
psyllama ps
```

### Stop a running model

```
psyllama stop gemma3
```

### Start Psyllama

```
psyllama serve
```

To view a list of environment variables that can be set run `psyllama serve --help`
